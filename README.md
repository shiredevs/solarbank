# Solarbank
## Quickstart
### Client
For first time setup navigate to the [client](./client) folder and run:
```shell
npm ci
```
To start the ui run:
```shell
npm run start
```
The ui will be served at http://localhost:3000.

To run the tests run:
```shell
npm run test
```
To build a static bundle:
```shell
npm run build
```
